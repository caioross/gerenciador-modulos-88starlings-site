Landing bilíngue (PT/EN) com tema espacial, explicando o app local (Three.js) que cataloga os módulos .glb da nave e exporta o `modules_catalog.json`.

## Rodar localmente

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # produção
```

## Stack

Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS · i18n client-side · Google Analytics 4.
